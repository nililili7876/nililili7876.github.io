Apprentissage par Renforcement

1. Rappels

Problème: 给定一个决策问题[problème décisionnel], 怎么找到最优策略?

Exemple 1 : Récompense terminale

Dans le cas de la bille sur le plateau, comment savoir quel coup jouer dans chaque case pour optimiser les chances de gain ?

Exemple 2 : Récompense continue

Dans le cas d’une machine à sous (Bandit manchot), comment trouver la bonne stratégie tout en maximisant son gain ?

CHAT乱说的一些

老虎机问题/Slot Machine Problem: 有一台赌博机，玩家每次投入一定的代币（或金钱），然后拉动手柄或按下按钮，赌博机会以一定的概率给出不同的奖励。问题的关键在于，玩家如何最大化自己的收益。

具体来说，老虎机问题可以被抽象为一个序列决策过程（Sequential Decision Process），其中玩家需要在每一轮中决定是否继续玩游戏，并且决定下一步的动作（拉动手柄或按下按钮）。每一次动作都会给出一定的收益，收益的大小是随机的，根据一定的概率分布来确定。玩家的目标是在有限的时间内最大化自己的总收益。是一个经典的强化学习问题，因为它涉及到探索（exploration）和利用（exploitation）的平衡问题。在初始阶段，玩家需要尽可能多地探索不同的动作，以确定不同动作的期望收益。但是随着时间的推移，玩家应该逐渐减少探索的频率，并更多地利用已知的信息，以最大化自己的总收益。

2. Apprentissage par Renforcement

Pourquoi l’apprentissage par renforcement ?

当模型不精确已知或模型过于复杂而无法进行符号分析时，允许即时计算最优策略

Beaucoup d’autres méthodes

  • Optimisation
  • Algorithmes génétiques: 遗传算法
  • Recuit simulé: 模拟退火

Apprentissage par Renforcement 考虑到 structure temporelle. ⇒ L’ordre des expériences est important.

Dilemme Exploration / Exploitation

是选择一个自己熟悉的好act/état，还是探索一个新act/état？

Plusieurs méthodes de choix possible,不定向的或定向[dirigées]

  • Constante : Choisir N fois la meilleure actions, puis M fois une action aléatoire.

    选择N次最佳action，然后M次随机

  • ε-greedy : Choisir la meilleure action avec probabilité 1 − ε, et une action aléatoire avec probabilité ε.

    选择概率为1-ε的最佳action，以及概率为ε的随机

  • Softmax : Tirer l’action selon une distribution de Boltzman (dépendant de la valeur des actions).

    根据Boltzman分布给action排序（取决于行动的价值）

  • Recency-based : bonus d’exploration fonction du nombre d’itérations depuis le dernier a.

    探索奖励，取决于自上一次a以来的迭代次数

  • Uncertainty estimation : bonus d’exploration selon le nombre de fois que a a été choisie.

    根据选择a的次数获得探索奖励

Algorithmes de calcul de V

我们用méthodes incrémentales[增量], 3 classes d’algorithmes :

  • Programmation dynamique : 先验的知道了模型的话

  • Monte Carlo : aucune connaissance a priori nécessaire, mais pas vraiment incrémentale.

    不需要先验的知道模型，但不是真正的增量。

  • Différence temporelle : utilisent une connaissance partielle du modèle, propagent les fonctions de valeur. ⇒ Le véritable apprentissage par renforcement.

    使用模型的部分知识，传播价值函数。⇒ 真正的强化学习

在第k+1次迭代，我们更新$$V_{k+1}^{\sigma}(s_j)\leftarrow V_k^{\sigma}(s)+\alpha[r_j+\gamma V_{k}{\sigma}(s_{j+1})-V_{k}{\sigma}(s_{j+1}]=V_k^{\sigma}(s_j)+\alphaδ_j$$

2.3 Algorithme SARSA

Fonctions d’action-valeur

adversaire是给定的，Q-fonctions: Qσ(s,a)=r(s,a)+γ∑s′P(s,a,s′)Vσ(s′)Q^{\sigma}(s,a)=r(s,a)+\gamma\sum_{s'}P(s,a,s')V^{\sigma}(s')Qσ(s,a)=r(s,a)+γ∑s′​P(s,a,s′)Vσ(s′)

  • Vσ(s)=Qσ(s,σ(s))V^{\sigma}(s) = Q^{\sigma}(s,\sigma (s)) Vσ(s)=Qσ(s,σ(s))

  • L’équation de Bellman: ∀s,a    Q∗(s,a)=r(s,a)+γ∑s′P(s,a,s′)maxbQ∗(s′,b)\forall s,a\ \ \ \ Q^{*}(s,a)=r(s,a)+\gamma\sum_{s'}P(s,a,s')max_bQ^*(s',b)∀s,a    Q∗(s,a)=r(s,a)+γ∑s′​P(s,a,s′)maxb​Q∗(s′,b)
  • 用得到的Q∗Q^*Q∗可以得到V∗V^*V∗和σ∗\sigma^*σ∗
    • V∗(s)=maxaQ∗(s,a)V^∗ (s) = max_a Q^∗ (s, a)V∗(s)=maxa​Q∗(s,a)
    • σ∗(s)=argmaxaQ∗(s,a)\sigma^∗ (s) = argmax_a Q^∗ (s, a)σ∗(s)=argmaxa​Q∗(s,a)

State-Action-Reward-State-Action (SARSA)

在第k+1次迭代，avec给定的adversaire

缺点：更新时我们必须知道下个état里的choix

2.4 Q-Learning

为了简化SARSA，不需要知道当前的adversaire，方法

在k+1词迭代，如果我们在état s_j选择了action a

Vk+1σ(sj,a)←Vkσ(s,a)+α[rj+γmaxbQk(sj+1,b)−Qk(sj,a]V_{k+1}^{\sigma}(s_j,a)\leftarrow V_k^{\sigma}(s,a)+\alpha[r_j+\gamma max_b Q_k(s_{j+1},b)-Q_{k}(s_{j},a] Vk+1σ​(sj​,a)←Vkσ​(s,a)+α[rj​+γmaxb​Qk​(sj+1​,b)−Qk​(sj​,a]

优点: 比SARSA更简单，收敛性更早被证实，是强化学习中使用最多的算法

缺点: 在实践中往往不如SARSA有效
