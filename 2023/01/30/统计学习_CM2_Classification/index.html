3. Multiclass and multilabel classification

  1. Multiclass
    • a classification task with more than two classes (e.g. several
      animals)==>非二元的分类
    • each sample is assigned to one and only one label: a fruit can be either an apple or a pear but not both at the same time.==>每个样本只有一个标签
  2. Multilabel
    • each sample is assigned to a set of target labels.
    • label = properties that are not mutually exclusive[互不排斥的属性], e.g. (topics of a collection of books)

如何用二元分类器解决多标签多类别的分类问题？==> One Versus All strategy

  • 对于每个class，只有是和不是这两种
  • Interpretability[可解释性] : since each class is represented by one and only one classifier, it is possible to gain knowledge about the class by inspecting its corresponding classifier.
  • The final decision corresponds to the classifier with the highest score

4.1 Confusion Matrix

False Positive (FP) rate： The fraction of negative examples that are classified as positive.==>被归类为正面的负面例子的比例。

False Negative (FN) rate：The fraction of positive examples that are classified as negative.

同理有True Positive (TP) and True Negative (TN)

4.4 Unbalanced classes in classification

就是一个东西可能即是这个class又是那个class，然后在这种情况下，classification rules tend to predict only the majority class : accuracy is not enough.

  • Rebalance the metric. Empirical risk with rebalanced loss

  • oversampling methods : creates copy data or create synthetic samples (SMOTE) from the minor class[小类]
