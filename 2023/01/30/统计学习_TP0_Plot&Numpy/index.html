1.1 plt VS subplots

A plot is mainly made of two objects: a figure and a set of axes.

  • Roughly speaking, the figure is the big box in which everything is put (it is the top level container). For instance, a figure is characterized by a size, a title and a facecolor.
  • The axes are the plots themselves[图本身].There can be several plots inside a figure (also called subplots) and they are, for example, characterized by ticks, limits on x and y, a legend and a title. The axes contain the curves plotted.

1.2 subplots

  • fig,axs = plt.subplots(2)==>ax[0].plot(x1,y1) 或者 ax1.plot(x,y)#注意是1
  • fig,axs = plt.subplots(2,2)==>ax[0,0].plot(x1,y1) 或者 ax1.plot(x,y)#注意是1
  • 遍历每个子图可以：for ax in fig.get_axes()
  • subplots里面可以加sharex=True这种

2.4 Histogram VS Bar

概念上

**直方图（Histogram）**是一种可视化在连续间隔，或者是特定时间段内数据分布情况的图表，经常被用在统计学领域。简单来说，直方图描述的是一组数据的频次分布直方图有助于我们知道数据的分布情况，诸如众数、中位数的大致位置、数据是否存在缺口或者异常值。

比较：

  • 直方图展示数据的分布，柱状图比较数据的大小。
  • 直方图X轴为定量数据，柱状图X轴为分类数据。
  • 直方图柱子无间隔，柱状图柱子有间隔
  • 直方图柱子宽度可不一，柱状图柱子宽度须一致

Numpy

1. 创建array：

  • 从列表：a = np.array(l)，可以加上dtype指定类型
  • np里面的一些函数: np.arange(15),np.zeros((3,4)),np.ones((2,10))：全1的矩阵,np.eye(3):单位矩阵，np.diag([1,2,3]):对角矩阵，np.linspace(0,1,6):注意与arange区分，那个是地点终点步长，这个是起点终点和个数

2.array的属性

a.dim:维数/行，a.shape,a.size:总数,a.dtype:类型

3.修改形状

  • reshape: m = np.arange(12); m.reshape(4, 3)
    • -1: 提供新形状的满足标准是“新形状应与原始形状兼容”,自动补全的未知形状==>比如m.reshape(-1, 1)就是把原来的变成列向量
  • ravel:变成一维数组

4.重复与连接

np.tile(m,(2,3)): 把m的行重复两次，列重复3次

np.concatenate((m,n)): 连接m和n，默认接着行，加axis=1表示横着列连接

对于二维数组，连接也可以用np.vastack[竖折]和np.hstack[横着]

5.索引和切片

正常取位置：m[3,2]或者m[3] [2]，m[3,:]:第三行, m[:,3]:第三列,m[[1,1,2],[0,1,2]]:取(1,0),(1,1)和(2,2)

8. Broadcasting

广播(Broadcast)是 numpy 对不同形状(shape)的数组进行数值计算的方式， 对数组的算术运算通常在相应的元素上进行。当运算中的 2 个数组的形状不同时，numpy 将自动触发广播机制。

Scipy

Scipy is a collection of mudules for various mathematical purposes. Leveraging the Numpy package, Scipy offers a broad and powerful environment for solving scientific problems. Roughly speaking, Scipy is to Python what toolboxes are to Matlab.

  • An important feature of the scipy.special module is the definition of numerous mathematical functions, called special functions. Available functions include Airy, elliptic, Bessel, gamma, beta, hypergeometric, parabolic cylinder, Mathieu, spheroidal wave, Struve, and Kelvin. ==> 很多数学函数的包
  • The scipy.linalg module is crafted for linear algebra. It subsumes[包括] all the functions in numpy.linalg and adds other more advanced ones.
    • 比如说可以解线性方程: x = linalg.solve(C, b)
    • 然后还有一些特殊矩阵的构造器：对角线矩阵、循环矩阵、Hadamard 矩阵、Toeplitz 矩阵……
  • The scipy.optimize module offers several common but efficient routines for constrained and unconstrained optimization, as well as curve fitting and root finding.
    • optimize.minimize
    • optimize.curve_fit
    • optimize.fsolve
  • scipy.interpolate provides functions for uni and multivariate data interpolation. Here, we only focus on unidimensional interpolation
  • In the scipy.integrate module, one can find several methods (trapezoidal, Simpson, Romberg, etc) for integrating uni, bi and trivariate functions.
