Introduction to Statistical Learning

统计学习的基本介绍和Supervised learning

1. What is Machine Learning ?

Machine Learning is the science of programming computers so they can learn from data.

example: spam, Handwritten digits recognition,fraud detection[诈骗识别], recommendations, text mining[挖掘]

competence:

  • Problems for which existing solutions require a lot of hand-tuning or long lists of rules : one Machine Learning algorithm can often simplify code and perform better.
  • Complex problems for which there is no good solution at all using a traditional approach (modelling) : hopefully efficient Machine Learning techniques can find a solution.

2. Types of ML problems

2.1 Supervised versus Unsupervised Learning

Supervised learning : regression, classification, structured Learning (e.g. predict trees or graphs)

Unsupervised : clustering[聚类], anomaly [遗产]detection

Semi supervised learning and transductive learning : all the labels are not observed.

2.2 Reinforcement learning

  1. A learning system takes actions (a policy) in an environment so as to maximize some notion of cumulative[累积] reward.
  2. Many robots implements Reinforcement Learning
  3. DeepMind’s AlphaGo program (game of Go)

2.3 Batch Learning versus Online Learning

Batch learning : the system is trained, and then it is launched into production and runs without learning anymore ; it just applies what it has learned.

Online learning : the system is trained incrementally by feeding it data instances sequentially, either individually or by small groups called mini-batches. Great for：

  • systems that receive data as a continuous flow ;
  • systems with limited computing resources ;
  • systems on huge datasets that cannot fit in one machine’s main memory

3. 概念对比

3.1 AI VS ML

Artificial intelligence (AI) is intelligence displayed by machines, in contrast with the natural intelligence (NI) displayed by humans and other animals.

Machine Learning is a subfield of AI : it is a field of computer science that gives computers the ability to learn without being explicitly programmed.

3.2 ML VS Statistical Learning

相同点：both fields focus on supervised and unsupervised problems

不同点：

  • Statistical learning is a subfield of Statistics.
  • Machine learning has a greater emphasis on large scale applications and prediction accuracy，Statistical learning emphasizes models and their interpretability[模型的可解释性], and precision and uncertainty.

4. Supervised learning

4.1 基本概念

X: vector of p measurements X (also called regressors, covariates[协变量], features).

Y: Outcome measurement Y (also called dependent variable, response, target).

  • In the regression problem, Y ∈ R is quantitative (e.g. price).
  • In the classification problem, Y takes values in a finite, unordered set[有限无序的离散集] (e.g. digit 0-9, image classification).
  • Structured prediction : Y is a more complex quantity : vector (multivariate regression) graph, tree, string, image …

问题：We have training data $D_n = {(X_1, Y_1), . . . ,(X_n, Y_n)}, (X_i,Y_i)$独立同分布的随机变量，We want to infer the link between the input variables X and the outcome Y,即 use Dn to build up a predictor f such that f(X) ≈ Y.

步骤:

  1. Data Collection
  2. Feature engineering (sampling, feature extraction, feature transformation …)
  3. Choose a statistical model (parametric or not) or an algorithm to find a learning rule f
  4. Fit/拟合 (statistics and optimization) the learning rule f or a collection of learning rules (f)λ∈Λ.
  5. Tune[调整] the parameters (typically by cross validation)
  6. Make prediction with the final model.

4.2 Generalization error & Bayes risk & Empirical risk

Risk of a predictor f also called generalization error or prediction error ：$R(f)=El(f(X),Y)$，$l(f(X),Y)$是损失函数，一些常见的损失函数

  • Quadratic loss : $l(a, b) = (a − b)^2$
  • Absolute loss : $l(a, b) = |a − b|$
  • 0-1 loss
  • Hinge loss :$l(a, b) = max(0, 1 − ab)$
  • Logistic loss :$l(a, b) = log[1 + exp(−ab)]$

Optimal risk or Bayes risk:$R^*=inf_{f\in F}R(f)$, F is the set of all possible predictors

Empirical risk: $R_n(f)=\frac1n\sum_{i=1}^nl(f(X_i,Y_i))$ ,是R(f)的unbiased estimator

4.3 Learning rule

  1. The risk is conditional to Dn: $R(f) = E[l(f(X),Y)|D_n]$

  2. Mean risk: $ER(f)= E[l(f(X),y)]$

  3. Empirical Risk Minimization (ERM): $min_{f\in F}\frac 1N\sum_{i=1}^NL(y_i,f(x_i))$

4.4 Discriminative vs Generative Methods

$y_{N+1}=arg max_yP(y|x_{N+1})$

Generative Approaches:

  • Based on the conditional distribution (X|Y) distribution and the Bayes Theorem
  • 基于联合分布P(X,Y)
  • Gaussians, Naive Bayes, Linear / Quadratic，Discriminant Analysis, Hidden Markov Models (HMM), Bayesian，networks, Markov random fields

Discriminative Approaches

  • Directly optimize over a class of predictor
  • 基于条件概率P(Y|X)
  • Linear regression, Logistic regression, SVMs,neural networks, Nearest neighbour, Conditional Random Fields(CRF), Random Forests, boosting

在机器学习中任务是从属性X预测标记Y，判别模型求的是P(Y|X)，即后验概率；而生成模型最后求的是P(X,Y)，即联合概率。

判别式模型举例：要确定一个羊是山羊还是绵羊，用判别模型的方法是从历史数据中学习到模型，然后通过提取这只羊的特征来预测出这只羊是山羊的概率，是绵羊的概率。

生成式模型举例：利用生成模型是根据山羊的特征首先学习出一个山羊的模型，然后根据绵羊的特征学习出一个绵羊的模型，然后从这只羊中提取特征，放到山羊模型中看概率是多少，在放到绵羊模型中看概率是多少，哪个大就是哪个。
