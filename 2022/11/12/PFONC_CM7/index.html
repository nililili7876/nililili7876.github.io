Maybe, muni de mfmap, est appelé un foncteur

Cette construction est généralisable pour d’autres constructeurs de type, elle permet de promouvoir (lift) une fonction de type a->b en une fonction de type m a->m b.

Haskell possède une classe de types Functor, dans laquelle la fonction de promotion s’appelle fmap, fmap doit vérifier les propriétés suivantes :

  • fmap id == id
  • fmap (f.g) == (fmap f).(fmap g)

Maybe, [], r-> ont des instances de la classe Functor.

fmap: (a->b) -> f a -> f b ,将盒子[Just,Maybe类型]里面的值取出来进行函数操作

<*> (apply函数)：f (a->b) -> f a -> f b,将盒子装的函数取出来与盒子中的值进行操作

bind函数>>=: Monad f => f a -> (a -> f b) -> f b, 接收一个值的盒子和一个返回值是盒子的函数，返回一个装着值[结果]的盒子
