Types de base

entiers: Integer/Int

Réels virgule flottante: Double ou Float

  1. Paire, True, False, Canard, etc. sont des constructeurs de données

    都是值构造子，值构造可以与类型名同名，后面可以跟几个类型表示它包含值的类型

  2. Ils peuvent avoir n’importe quel nom, y compris le même que le nom du type défini.Comme pour Entiers3 ; [可以同名]

  3. Les noms de types et de constructeurs commencent par une majuscule[大写字母] en Haskell.然后普通函数必须小写字母开头

Types en Haskell

type et newtype

-type définit un type synonyme[同义] d’un autre：type String = [Char]

-newtype est similaire à data mais ne fonctionne que pour un seul constructeur de données avec au plus une variable et fournit dans ce cas une petite optimisation (Synonymes mais crée un type distinct).

但只适用于最多只有一个变量的单一数据构造函数
